# KUAIZAI_OS 应用图标

本目录包含应用打包所需的各种格式图标。

## 图标规格

- **icon.ico** - Windows 可执行文件图标 (256x256 多尺寸)
- **icon.icns** - macOS 应用图标 (512x512)
- **32x32.png** - 任务栏图标
- **128x128.png** - 桌面快捷方式图标
- **128x128@2x.png** - 高分辨率桌面图标

## 生成图标

使用 Tauri 图标工具从 1024x1024 的源图生成：

```bash
# 在项目根目录执行
npm run tauri icon app-icon.png
```

源图要求：
- 尺寸：1024x1024 像素
- 格式：PNG
- 透明背景
- 正方形设计

## 占位符图标

当前使用占位符图标，正式版本应替换为实际设计的应用图标。
