# 🌐 AI Game Companion Assistant - 网页版

纯前端实现的网页版本，可直接在浏览器中运行。

## ✨ 功能特性

- 💬 **实时对话** - 与 AI 助手进行游戏相关对话
- 📤 **图片上传** - 点击上传游戏截图进行分析
- 🎭 **毒舌模式** - 独特的 AI 人格，幽默讽刺但有帮助
- 📱 **响应式设计** - 适配不同屏幕尺寸
- ⚡ **快速启动** - 无需安装，浏览器直接打开

## 🚀 快速开始

### 方法一：双击启动（推荐）

直接双击 `启动.bat` 文件，会自动启动本地服务器并打开网页。

### 方法二：手动启动

如果你已安装 Node.js：

```bash
npx serve -l 8080 -s
```

或使用 Python：

```bash
python -m http.server 8080
```

### 方法三：使用 VS Code

安装 Live Server 插件，右键 `index.html` 选择 "Open with Live Server"。

## 🌐 访问地址

启动后，在浏览器中打开：

- 本机访问：**http://localhost:8080**
- 局域网访问：`http://你的IP地址:8080`

## 🔧 服务器配置

网页版默认连接以下服务器：

```javascript
remoteApiUrl: 'http://100.122.65.26:8000'  // Tailscale 虚拟 IP
```

### 代理设置

如果你的系统使用了 HTTP 代理，可能无法直接连接 Tailscale 虚拟 IP。

**解决方案：**

1. **临时关闭代理运行**
   ```cmd
   set http_proxy=
   set https_proxy=
   启动.bat
   ```

2. **配置代理排除规则**
   - 将 `100.122.65.26` 添加到代理排除列表

## 📝 使用说明

### 图片上传分析

1. 在对话页面，点击输入框左侧的 **图片图标** 📎
2. 选择要上传的图片文件（支持 JPG、PNG、GIF，最大 10MB）
3. 可选：在输入框中添加描述文字
4. 点击「发送图片」按钮
5. AI 将分析图片内容并回复

### 支持的图片格式

- JPG / JPEG
- PNG
- GIF
- WebP

**限制：** 单张图片最大 10MB

## 🔨 常见问题

### Q: 无法连接服务器？

**检查步骤：**
1. 检查 Tailscale 是否连接
   ```cmd
   tailscale status
   ```

2. 测试服务器连通性
   ```cmd
   curl http://100.122.65.26:8000/health
   ```

3. 检查是否被代理拦截
   - 尝试关闭代理后刷新页面

### Q: 页面显示空白？

1. 确保使用本地服务器打开（直接双击 index.html 可能无法加载 JS）
2. 检查浏览器控制台（F12）是否有报错

### Q: 图片上传失败？

1. 检查图片格式是否支持
2. 检查图片大小是否超过 10MB
3. 检查网络连接是否正常
